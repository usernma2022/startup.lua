# startup.lua
This code is for ComputerCraft, Lua on Minecraft. The code is freely accessible for you to use. However, I do not provide mods for these to work. Thanks.

Scripts:
TurretHandler - kinda doesnt work, working on it.
RadarHandler - Finds players, currently compatible with Logs, no main file. Also sets up a screen and comes with chat alerts.
PeripheralTester - Allows you to see exactly what a CC computer can configure if theres a block next to it. Outputting nothing means theres nothing to be configured by CC.
Monitorfix - Fixes 3x3 computers to smallest size, makes radars work.
Main HQ Script - Planned to take data from logs, damages to doors, and allows configuration of facility status. Basically the main system that tries to control as much power as you give it.

The rest of the scripts are for testing the main scripts.
